# react-d3-cloud
A word cloud react component using d3-cloud

## usage
```
npm install react-d3-cloud
```
```
import WordCloud from 'react-d3-cloud'

const data = {
  word1: 100,
  word2: 200,
}
<WordCloud data={data} />
```

## build
```
npm run build
```

## Test
